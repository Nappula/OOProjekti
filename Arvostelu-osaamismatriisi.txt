Koska valitsin alunperin "työkaluksi" 2d listan, en halunnut pyörtää päätöstä tältä osin, vaan halusin opetella käyttämään tätä.
Ongelmana "työkalun" (2d lista) oli etten projektin alussa vielä tiennyt mitä en tiedä, joten projekti eteni virheestä virheeseen.


Alla osaamismatriisi kirjoitettuna auki, pääset tutkimaan ehdotustani osaamismatriisin kohdista tarkastelemalla Task listiä (ctrl+§  +  ctrl +T).
Kohdat mitkä mielestäni täyttää osaamismatriisin kohdat on merkitty //ARVOSTELU tägillä, mikäli tägien asetukset siirtyvät solutionin mukana, näkyvät nämä suoraan taskListissa

https://docs.microsoft.com/en-us/visualstudio/ide/using-the-task-list?view=vs-2019



o = toteutuu osaamismatriisin vaikeimmassa arvostelussa.
X = kohta osoitettu TaskListissä


Oliot:
	X 1: Osaa luoda luokasta olion.
	X 2: osaa luoda dynaamisen oliokokoelman.
	X 3. osaa määritellä oliolle kentät ja metodit.
	X 4. osaa määritellä näkyvyydet julkisiksi ja yksityisiksi.
	X 5. Osaa määritellä oliolle property -kenttiä.

Luokkarakenne:
	o 1: osaa määritellä luokan, jossa on konstruktori 
	o 2: osaa määritellä luokan, jossa on viite toiseen olioon 
	o 3: osaa määritellä luokan, jossa on viite oliokokoelmaan 
	X 4: osaa määritellä luokan, joka perii yläluokan
	X 5: osaa määritellä luokan, jossa hyödynnetään metodin ylikirjoittamista 

Rajapinnat:
	o 1: osaa määritellä rajapinnan
	o 2: osaa määritellä rajapinnan toteuttavan luokan
	X 3: osaa hyödyntää polymorfismia rajapinnan avulla
	o 4: lähdekoodista on erotettu jokin osa on rajapinnan avulla
	X 5: lähdekoodista on erotettu useampi osa rajapintojen avulla

Dokumentointi:
	x 1: Lähdekoodi on luettavaa
	x 2: Lähdekoodi on jäsennelty // luokissa sama rakenne, private fieldit, public fieldit, propertyt, konstruktorit, metodit.
	x 3: lähdekoodista on tehty luokkakaavio
	x 4: lähdekoodissa on kommentteja: //sekä inline kommentteja että ///<summary> kenttiä.
	x 5: lähdekoodi on itsekommentoivaa // muuttujat on nimetty kohtuu järkevästi

Tallentaminen:
	o 1: osaa tallentaa tekstiä tiedostoon
	o 2: osaa ladata tekstiä tiedostosta
	X 3: osaa tallentaa oliodatan tiedostoon
	X 4: osaa ladata oliodatan tiedostosta
	X 5: osaa hyödyntää oliodatan sarjallistamista
